# Venn Email Python

## Setup

- [virtualenvwrapper](https://virtualenvwrapper.readthedocs.org/en/latest/)

```
git clone https://github.com/VennHQ/venn-email-node
cd venn-email-node
mkvirtualenv -a . -r requirements.txt venn-email-node
```

## Quick start

```
workon venn-email-node
python example.py
```
